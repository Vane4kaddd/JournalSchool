# Школьный журнал на Flask

Это веб-приложение школьного журнала, созданное с использованием Flask. Оно поддерживает расписание, оценки, домашние задания, регистрацию пользователей по ролям (ученик, учитель, администратор) и другие функции.

## Функции

- Регистрация и вход по ролям
- Управление расписанием и домашними заданиями
- Система оценок
- Уведомления
- Поддержка тёмной темы
- Родительский контроль (доступ к аккаунту ребёнка)

## Структура проекта

```
├── journal.py            # основной файл приложения
├── requirements.txt      # список зависимостей
├── Procfile              # команда запуска на Render
├── templates/            # HTML-шаблоны
├── static/               # CSS, JS, изображения
└── instance/             # конфигурационные и приватные файлы (например, база данных)
```

## Установка

1. Клонируйте проект:
   ```bash
   git clone https://github.com/Vane4kaddd/SchoolJournal.git
   cd SchoolJournal
   ```

2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # или venv\Scripts\activate на Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите сервер:
   ```bash
   flask run
   ```

## Развёртывание на Render

1. Создайте репозиторий на GitHub и залейте туда проект.
2. Перейдите на [render.com](https://render.com), создайте аккаунт.
3. Создайте новый Web Service → подключите GitHub → выберите репозиторий.
4. Настройте параметры:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn journal:app`
   - **Runtime**: Python
5. Укажите переменные окружения, если нужно (например, `FLASK_ENV=production`).
6. Нажмите Deploy.

## Заметки

- `Procfile` должен находиться в корне проекта.
- Приложение должно экспортировать `app` в `journal.py`.
- Для работы на Render убедитесь, что база данных не в `.gitignore` (если SQLite) или используйте внешнюю (например, PostgreSQL).
